## ツンあり | 感謝カード作成アプリ（ツンデレ言葉特化版）
https://www.tun-ari.com/

## サービス概要  
感謝の言葉を素直に言いたいけど言えない人に    
笑いを交えて  
感謝を伝えるサービスです。  
特にツンデレ言葉に特化して、作成しやすくなっています。  

## 使い方  
 　　　①キャラクターを選ぶ 　 　　　　　②文字を入力する  　　 　　　　　   ③投稿のいいね
